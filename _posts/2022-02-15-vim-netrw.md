### 打开文件浏览器
使用:Explore命令或缩写:E命令（注意E大写），将在当前窗口中打开文件浏览器

### 新建文件
```
%
```
### 重命名
```
R(大写)
```
### 删除文件或目录
```
D
```
### 创建目录
```
d
```
### 新建窗口打开文件
```
o
```
### 切换显示方式
```
i
```
### 在标签页中打开
```
t
```

### 在后台标签页中打开
```
T 
```
