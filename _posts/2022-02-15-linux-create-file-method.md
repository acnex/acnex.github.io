# 1. vim

```
$ vim [file name]
$ i
$ [Document content]
$ Esc
$ :
$ wq Enter
```


# 2. dd

```
$ dd of=[file name]
$ [Document content]
$ Ctrl + d
```




# 3. touch

```
$ touch [file name]
```
